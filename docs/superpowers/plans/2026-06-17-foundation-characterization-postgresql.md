# Foundation, Characterization, and PostgreSQL Development Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Establish a safe, reproducible engineering baseline around the current SIST-iONM application, capture its observable behavior, audit the legacy SQLite database, and provide a healthy PostgreSQL development service without changing production behavior yet.

**Architecture:** This is phase 1 of the approved modular-monolith migration. The legacy `app/main.py` remains the executable application while new `app/core` infrastructure and tests are introduced beside it; PostgreSQL is started and verified, but runtime cutover occurs in phase 2 after the relational schema and migration pipeline exist.

**Tech Stack:** Python 3.13, FastAPI 0.137.1, Uvicorn 0.49.0, Pydantic Settings 2.14.1, pytest 9.1.0, Ruff 0.15.17, mypy 2.1.0, PostgreSQL 18.2, Docker Compose 5.

## Global Constraints

- Preserve all existing URLs, templates, and user-visible flows.
- Use `SIST-iONM_V2_6_CHAT_BITRIX_NOTIFICACOES` as the source version.
- Target Linux/Ubuntu deployment with Docker Compose.
- PostgreSQL is the final runtime database; SQLite remains read-only migration input.
- Never commit `.env`, databases, uploads, exports, backups, virtual environments, password hashes, or credentials.
- Production Python changes follow red-green-refactor; configuration-only files are verified with their native validation commands.
- Use Python `>=3.13,<3.14` in this phase.
- Do not move routes or templates in this phase.
- Run commands from the repository root unless a step states otherwise.

---

## Plan Decomposition

The approved specification spans independent subsystems, so implementation is split into reviewable plans:

1. **This plan:** repository baseline, dependency lock, settings, characterization tests, SQLite audit, and PostgreSQL development service.
2. **PostgreSQL schema and cutover:** SQLAlchemy models, Alembic baseline, SQLite-to-PostgreSQL loader, validation report, and transitional runtime gateway.
3. **Identity and security:** sessions, Argon2 migration, CSRF, authorization dependencies, users, roles, profiles, and protected downloads.
4. **Catalog and CRM:** suppliers, products, clients, opportunities, opportunity documents, and related templates.
5. **Sales and finance:** orders, closings, commissions, receivables, payables, costs, and atomic financial transactions.
6. **Portal and chat:** feed, organization chart, rooms, messages, notifications, authenticated WebSockets, and private uploads.
7. **Reporting, administration, and deployment:** PDFs, BI, seller reports, settings, backup/restore, Nginx, Ubuntu runbook, and final file map.

Each later plan is written and reviewed immediately before its implementation so its interfaces are based on verified code from the preceding phase.

```python
from enum import StrEnum
from functools import lru_cache
from pathlib import Path
from typing import Self

from pydantic import SecretStr, model_validator
from pydantic_settings import BaseSettings, SettingsConfigDict


class Environment(StrEnum):
    DEVELOPMENT = "development"
    TEST = "test"
    PRODUCTION = "production"


class Settings(BaseSettings):
    model_config = SettingsConfigDict(
        env_file=".env",
        env_file_encoding="utf-8",
        env_prefix="SIST_IONM_",
        extra="ignore",
    )

    app_name: str = "SIST-iONM"
    environment: Environment = Environment.DEVELOPMENT
    debug: bool = False
    timezone: str = "America/Sao_Paulo"
    database_url: str
    session_secret: SecretStr
    smtp_encryption_key: SecretStr
    storage_root: Path = Path("storage")
    legacy_sqlite_path: Path = Path("data/overpriceon_web.db")

    @model_validator(mode="after")
    def validate_secrets(self) -> Self:
        if len(self.session_secret.get_secret_value()) < 32:
            raise ValueError("session secret must contain at least 32 characters")
        if len(self.smtp_encryption_key.get_secret_value()) < 32:
            raise ValueError("SMTP encryption key must contain at least 32 characters")
        return self


@lru_cache(maxsize=1)
def get_settings() -> Settings:
    return Settings()
```

Create empty `scripts/__init__.py`, then create `scripts/audit_sqlite.py`:

```python
import argparse
import json
import sqlite3
from dataclasses import asdict, dataclass
from pathlib import Path


@dataclass(frozen=True)
class TableAudit:
    name: str
    row_count: int
    columns: tuple[str, ...]
    foreign_key_count: int


@dataclass(frozen=True)
class DatabaseAudit:
    path: str
    size_bytes: int
    integrity: str
    tables: tuple[TableAudit, ...]

    def to_json(self) -> str:
        return json.dumps(asdict(self), ensure_ascii=False, indent=2)


def audit_sqlite(path: Path) -> DatabaseAudit:
    resolved = path.expanduser().resolve()
    if not resolved.is_file():
        raise FileNotFoundError(resolved)

    uri = f"file:{resolved.as_posix()}?mode=ro"
    with sqlite3.connect(uri, uri=True) as connection:
        integrity = str(connection.execute("PRAGMA integrity_check").fetchone()[0])
        table_names = [
            str(row[0])
            for row in connection.execute(
                """
                SELECT name
                FROM sqlite_master
                WHERE type = 'table' AND name NOT LIKE 'sqlite_%'
                ORDER BY name
                """
            )
        ]
        tables = tuple(
            TableAudit(
                name=name,
                row_count=int(
                    connection.execute(f'SELECT COUNT(*) FROM "{name}"').fetchone()[0]
                ),
                columns=tuple(
                    f"{row[1]}:{row[2] or '?'}"
                    for row in connection.execute(f'PRAGMA table_info("{name}")')
                ),
                foreign_key_count=sum(
                    1 for _ in connection.execute(f'PRAGMA foreign_key_list("{name}")')
                ),
            )
            for name in table_names
        )

    return DatabaseAudit(
        path=resolved.name,
        size_bytes=resolved.stat().st_size,
        integrity=integrity,
        tables=tables,
    )


def main() -> int:
    parser = argparse.ArgumentParser(description="Audit a SQLite database in read-only mode")
    parser.add_argument("database", type=Path)
    parser.add_argument("--output", type=Path)
    args = parser.parse_args()

    report = audit_sqlite(args.database)
    payload = report.to_json()
    if args.output:
        args.output.parent.mkdir(parents=True, exist_ok=True)
        args.output.write_text(payload + "\n", encoding="utf-8")
    else:
        print(payload)
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```
